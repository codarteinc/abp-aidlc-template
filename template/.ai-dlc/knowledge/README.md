# Knowledge artifacts (synthesized on first elaboration)

This directory is empty in fresh scaffolds. The first run of
`/ai-dlc:elaborate` for a substantive intent will synthesize
knowledge artifacts here (`architecture.md`, `conventions.md`,
`design.md`, `domain.md`, `product.md`) by reading your codebase.

You can also manually seed any artifact ahead of time if you have
existing docs to bring in.

See https://github.com/codarteinc/linkhub for a worked example of
populated knowledge artifacts.
